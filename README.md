Minecraft-CTF 游戏规则
1. 目标与胜负
- 玩法是占点夺旗：把己方旗帜放到己方半场的金块点位上。
- 地图中每队最多有 8 个目标点（固定图是预设 8 点，随机图也生成 8 点）。
- 任意一方先把 8 个点全部插上己方旗帜即立即获胜。
- 比赛时长上限 3 分钟（3*60*20 ticks）。超时后按已占领点数判定胜负；点数相同则平局。

2. 阵营与分组
- 左队与右队由玩家名中的组号前缀识别（CTF-组号-xxx）。
- 进入对局后，系统会按组号把玩家分到左右队。
- 旁观/非参赛玩家可进世界，但在比赛中会被视为 NONE 队伍，不参与胜负。

3. 对局流程
1. 创建对局（手动菜单或聊天匹配）。
2. 进入准备阶段：玩家为创造模式、清理背包。
3. 参赛者发送聊天 I'm ready! 表示就绪。
4. 满足人数且双方都就绪后进入正式比赛。
5. 开局会重置地图并生成旗点/目标点/障碍与生物。
6. 达成胜利条件或超时后结束，保存录像并上传。

4. 地图与模式
- 地图范围核心区 x: -24..24, z: -36..36。
- 地图类型：
  - 固定地图（旗点和目标点按固定坐标生成）
  - 随机地图（左右半场各随机生成 8 个旗点 + 8 个目标点）
- 地图会额外生成树、瓶子草、哞菇、发光鱿鱼（哞菇含追踪冲撞 AI）。

5. 夺旗与交付规则
- 玩家靠近敌方旗座（地面铜块+栅栏+旗帜）会自动“拔旗”：
  - 旗座被清空为草地
  - 旗帜放到玩家头盔位（蓝/红旗）
- 玩家携旗走到己方半场任意空的金块目标点上会自动插旗成功。
- 插旗后头盔旗帜消失，目标点上出现对应队伍旗帜。

6. 抓捕、死亡、坐牢
- 玩家与敌方玩家碰撞时：
  - 若你在己方半场且敌人在你这边，敌人被抓进敌方监狱。
  - 若你在敌方半场且敌人也在敌方半场，你会被抓进己方监狱。
- 被抓或死亡都会掉旗（若正在携旗）。
- 死亡重生点固定在己方监狱。

7. 掉旗与回场
- 携旗玩家被抓/死亡时会在附近可放置位置重新立旗（优先搜索草地+上方两格空气）。
- 只能在左右半场内掉旗，不会掉到场外。

8. 玩家限制与保护
- 参赛者比赛中为冒险模式，并自动穿带绑定诅咒的队伍颜色皮革护甲。
- 食物值最低被钳制为 7（不会低于 7）。
- 物品耐久损耗被取消。
- 全局掉落物生成被取消（防止散落物影响）。
- 玩家移动会被边界与区域规则限制，无法跑出比赛区域。

9. 匹配规则（聊天指令）
- 聊天格式：with <team> <count> <map>
  - team: none / random / 具体数字组号
  - count: 每队人数
  - map: fixed / random / unspecified
- 含义：
  - none: 自建空对手房间
  - random: 随机匹配符合条件的队伍
  - 数字组号：指定对手组号

新手友好操作指南


A. 开打前你要知道的 3 件事
1. 你的名字必须符合分组格式（例如 CTF-1-Alice），系统才会识别队伍。
2. 开局前在聊天输入 I'm ready!，否则比赛可能不会开始。
3. 你的任务不是击杀数，而是“把旗子送到己方金块点位”。
  
B. 最快开局流程
1. 进入服务器后先到主世界。
3. 用聊天开一局测试（按T键进入聊天）
  - with none 1 fixed（无需匹配对手，固定地图）
4. 传送进比赛世界后，等待准备提示。
5. 在聊天发送：I'm ready!。
6. 开局后冲向敌方旗座，拔旗并撤回己方半场。
7. 找己方金块空点位站上去完成插旗。
8. 重复直到己方占满 8 点，或时间结束比点数。

C. 常见问题
- Q: 我为什么不能离开某个区域？
  - A: 插件有强制边界和阶段区域限制，防止越界与卡图。
- Q: 我死了为什么复活在监狱？
  - A: 这是规则设计，死亡即回己方监狱。
- Q: 我的旗子去哪了？
  - A: 被抓/死亡会掉旗，旗子会在附近合法位置重新立起。
- Q: 为什么比赛突然结束？
  - A: 可能是某队占满 8 点、时间到、或管理员手动结束。

Python基础
1. 变量 & =

name = "Tom"   # 字符串

👉 = 表示“赋值”（把右边的值放进左边）

2. 字符串 str
name = "Alice"
print(name)            # 输出 Alice
print("Hello " + name) # 拼接

👉 字符串就是文本，用 " " 或 ' ' 包起来

3. 条件判断 if
block = "gold_block"
if block == "gold_block":
    print("target")
else:
    print("unknown block:", block)
👉 : 必须写 , 缩进很重要（通常4个空格） 

4. 循环 while

2

 条件为 True 就一直循环 
 一定要让变量变化，否则会死循环！
